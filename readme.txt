Plugin generates a mailing platform, mainly for organizational info newsletters.

== Description ==

A rather simple mailing platform. Creates a new post type, which are then used to generate a newsletter email with multiple subjects and messages.


== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set the settings in the Tiedotettavat->asetukset menu for the plugin
